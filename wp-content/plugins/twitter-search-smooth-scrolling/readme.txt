== Description ==

This widget shows tweets in real-time that match a given search term. The search term can be changed in the options screen of the widget. Links or Mentions are changes so they become actual links. The tweets show using a smooth scrolling and easing effect. All animation parameters can be changed via the options screen.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget on the your Blog via Appearance -> Widgets4
4. Change the settings on the options form and save.
